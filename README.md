## Tiny pastebin

A tiny pastbin app built with Golang and friends.

copy from [simple-pastebin](https://github.com/yuxiaoy1/simple-pastebin)

### Dependence

- Golang
- Postgres

### How to run locally

Clone the repo locally and run follow:
```bash
$ createdb paste

$ cd tiny-pastebin

$ go mod tidy && go run .
```
